# tool

#### 项目介绍
日常工具类,主要是封装了一些日常会使用的常用工具方法。


#### 使用说明

1. 由于未上传到maven的中央仓库，故需要自己下载源码生成jar
2. maven的引入坐标为:

		<dependency>
			<groupId>com.yishui.common</groupId>
			<artifactId>tool</artifactId>
			<version>${版本号}</version>
		</dependency>

接下来就可以使用对应的工具类了
